Insertion Sort

Insertion sort is an algorithm that computes the sort in a really close way for the human being, and it can be used easily for sorting a deck of cards randomly. It requires $${O(n^2)}$$ operations to sort a list of size n (in the worst case).

At the beginning it has only one element, which is obviously the ordered set. After that, it has k ordered elements from lowest to highest, and it takes the k+1 element and it compares with all of the already ordered elements, stoping when it finds a lower element (all of the higher elements have been moved one right position) or when there is no elements (all of the elements were moved and this one is the lowest). At this point, the k+1 element gets inserted due to the other elements move.

public class InsertionSort{
  public static void ordenacion(Integer[] numeros) {
      for(int i = 1; i < numeros.length; i++){
        int value = numeros[i];
        int j = i - 1;

        while(j >= 0 && numeros[j] > value){
          numeros[j + 1] = numeros[j];
          j = j - 1;
        }
        numeros[j + 1] = value;
      }
  }
}
